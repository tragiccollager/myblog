# My Blog

这是我的个人博客网站，用于分享技术文章、生活感悟和学习笔记。

## 📝 功能特性

- 🏠 首页展示最新文章
- 📚 文章分类与标签
- 🔍 文章搜索功能
- 📱 响应式设计，支持移动端
- 🌙 深色/浅色主题切换
- 💬 评论系统
- 📊 文章阅读量统计

## 🚀 快速开始

### 安装依赖

```bash
# 克隆项目
git clone <your-repo-url>
cd myblog

# 安装依赖（根据实际技术栈补充）
npm install
```

### 本地开发

```bash
# 启动开发服务器
npm run dev
```

### 构建部署

```bash
# 构建生产版本
npm run build
```

## 📂 项目结构

```
myblog/
├── src/              # 源代码目录
├── public/           # 静态资源
├── docs/             # 文档
├── README.md         # 项目说明
└── .gitignore        # Git 忽略配置
```

## 📝 写作规范

1. 文章存放于 `posts/` 目录
2. 使用 Markdown 格式编写
3. 文件名格式：`YYYY-MM-DD-title.md`
4. 文章头部包含 YAML 前置信息
